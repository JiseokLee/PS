# 1513 : 지그재그 배열 3
  
## 문제 설명    
하나의 정수 n을 입력받아 n*n사각형에서 다음과 같은 삼각형 모양으로 지그재그로 출력하시오.

예를 들어, n이 5이면,

0 0 0 0 5
0 0 0 4 6
0 0 3 7 12
0 2 8 11 13
1 9 10 14 15
를 출력한다.

## 입력
정수 n이 입력된다.( 1 <= n <= 99)

## 출력
n*n사각형에서 삼각형 모양으로 지그재그 출력한다.

## 입력 예시   
5

## 출력 예시
0 0 0 0 5 
0 0 0 4 6 
0 0 3 7 12 
0 2 8 11 13 
1 9 10 14 15
