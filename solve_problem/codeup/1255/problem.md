# 1255 : 두 실수 사이 출력하기

시간 제한: 1 Sec 메모리 제한: 128 MB

## 문제 설명

소수 둘째 자리의 두 실수 a와 b가 입력으로 주어진다.

a와 b사이의 수를 0.01간격으로 오름차순으로 출력하시오.

예)

5.67 5.73 ==> 5.67 5.68 5.69 5.70 5.71 5.72 5.73

## 입력

두 실수 a와 b가 입력된다. (a <= b)

## 출력

a와 b사이의 수를 0.01간격으로 오름차순으로 출력하시오.

## 입력 예시

2.00 2.03

## 출력 예시

2.00 2.01 2.02 2.03
