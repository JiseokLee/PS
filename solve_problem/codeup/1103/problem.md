# 1103 : 폴더명 출력 해결
  
## 문제 설명      
printf()함수를 공부하는 진영이는 다음 폴더명을 출력해보기로 했다.

"c:\test"
printf함수에는 특별한 제어문자가 몇개 있다.

큰따옴표(")를 출력하기 위해선 \" 를 사용해야 하고,

\를 출력하기 위해선 \\ 를 사용해야 한다.

---- 웹상의 소스를 복사해서 붙이는 경우 에러가 날 수 있습니다.

## 입력
입력은 없다.

## 출력
"c:\test"

를 그대로 출력한다.(큰따옴표도 출력해야함)

## 출력 예시
"c:\test"
