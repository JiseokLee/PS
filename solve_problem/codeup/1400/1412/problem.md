# 1412 : 알파벳 개수 출력하기
  
## 문제 설명    
어떤 영어 문장이 주어지면 각 알파벳이 몇 번 나왔는지 출력하시오.

## 입력
영어 한 문장이 입력된다. (90글자 이내) 

이 문장은 영어 소문자와 공백, 특수문자로 이루어져있다.

## 출력
각 알파벳의 개수만 출력한다. 그외 특수문자나 공백의 개수는 출력하지 않는다.

oh! my god!
